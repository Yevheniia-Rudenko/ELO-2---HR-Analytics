# Notes

This folder is for shared team notes, resources, and learning materials.

## Contents

- Research notes
- Learning resources
- Useful links and references
- Project insights
- Technical documentation

## Organization Tips

- Create separate markdown files for different topics
- Use clear, descriptive filenames
- Add dates to time-sensitive notes
- Link related documents

## Suggested Files

- `research_notes.md` - Background research and findings
- `useful_resources.md` - Links to tutorials, articles, datasets
- `glossary.md` - Domain-specific terminology
- `troubleshooting.md` - Common issues and solutions
